### Clone the Project

To clone the project, run:

```bash
git clone https://github.com/yourusername/your-repo-name.git
```

### Navigate to Project Directory

cd your-repo-name

### Install Dependencies

npm install

### Start the Application

npm start

## Application URL

The server runs on:

```web
http://localhost:3000

```

### Build the Application

npm build
